First Team Meeting

  1. Select a project manager   Note: I may reject your choice.  The person you choose should have a proven record of being in class by 9:00 AM each morning, and have good organizational and communication skills.
  2. Select a git manager.  Note: I may reject your choice.  The person you choose should have exceptional git skills and have the ability to quickly manage pull requests from multiple users, including walking team members through conflict resolution.
  3. Optional - assign roles to other people
    • Team members will contribute to all aspects of project, but some team members may be assigned other specific roles: server management, database management, etc., if your team decides this is worth doing.
  4. What is your project?
    • Prepare an elevator pitch that your designated project manager can explain to me this morning.
  5. Get started setting up your server.
    • installation of an http server (apache or nginx, mysql or maria, php, php modules, security, additional access, etc.)
  6. Setup a VHOST for your project (your domain has been assigned)

Roles and Responsibilities

These are some of the common project roles.  It is not a requirement to assign specific people to specific roles (other than Project Manager and Git Manager), but if you do, common duties and responsibilities can be found below.  Note: it's also possible that one person may be assigned multiple roles.  For example, it's not uncommon for the Project Manger to also be the Git manager... but your selection of these two roles must be approved by me.

Project Manager - This person will meet with me for a few minutes every morning to update me on team progress.

  • Morning meeting (timesheets, git logs, etc)
  • Gathering timesheets
  • Assigning tasks (who has been assigned what work for the day?)
  • Tracking progress (how is yesterday's assigned work progressing?)
  • Programming and other duties as required

Server - This person will be the lead person in charge of the server, software installation, setup and security.

  • Install Apache, Mysql/Maria, PHP
  • Setup and configure firewall
  • Setup and configure project VHOST
  • Configure all file permissions in /var/www/project_folder
  • Programming and other duties as required

Database - This person will be the lead person in charge of designing the database, its tables, fields, data types, etc.

  • Initial ERD
  • Migrations or Creation Scripts
  • Seeders or Initial Insert scripts
  • Table alterations as required
  • Helping team members update their database if required
  • Programming and other duties as required

Git manager - This person will be the lead person in charge of programming

  • Manage git repository 
  • Test and either reject or merge pull requests
  • Ensure team follow correct git procedures
  • Mentoring/assisting other team members
  • Programming and other duties as required

Design and HTML - Initial graphic design of site in Photoshop or other design tool, based on discussions, recommendations, and input of team.

  • Initial design presentation and changes
  • Coding HTML templates from design
  • Conversion of templates to views
  • Layout and design of proposal or proposal elements
  • Programming and other duties as required
