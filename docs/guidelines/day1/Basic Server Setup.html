Basic Server Setup

    Update and Upgrade installed software

        sudo apt update
    sudo apt upgrade

    Install Apache, PHP and MySQL Server

        sudo apt install apache2 php mysql-server (or... sudo abpt install apache2 php mariadb-server)
    sudo apt install php-mbstring php-xml php-zip php-mysql php-sqlite3 php-gd php-zip php-curl
    sudo systemctl restart apache2

    Test everything works

    • create info.php in /var/www/html and visit. Confirm PHP version

    Anything else you want to do.

    eg:

    • Install composer
    • Install git
    • Install node
    • Install zip/unzip
