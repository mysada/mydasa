eCommerce Teams

When a team member is assigned a role by the team, it does not mean that that person is soley responsible for all work related to that role. It simply means that the person assigned the role will be the person that I talk to if I have questions.

All team members must contribute to every phase of the project.

Important skills

  • Project Management (assign work, manage process, reporting)
  • Git (managing pull requests and merges of team members)
  • Server (setup from barebones server provided to you)
  • MySQL (setup and keep database current)
  • Programming (problem solving, mentoring, coaching)
  • Design (graphic design and UX)
  • HTML/CSS/Javacript (Front end construction and enhancement)

Elements to be created in first few days

  • Proposal
  • Data Model
  • Data gathering … realistic
  • Server setup (PHP, MySQL, Apache, Security, SSL Certificate)
  • Graphic Design (Home Page, Generic Sub-page, List View, Detail View)
  • HTML and CSS templates of designed pages
